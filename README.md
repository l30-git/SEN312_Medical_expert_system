# SEN312 Medical Expert System

This project is a rule-based medical diagnostic expert system developed in Python.

## Features
- Diagnoses Malaria
- Diagnoses Typhoid
- Diagnoses COVID-19
- Diagnoses Pneumonia
- Diagnoses Dengue Fever
- Uses forward chaining inference
- Provides confidence scores
- Gives medical advice

## How to Run

```bash
python medical_expert_system.py
